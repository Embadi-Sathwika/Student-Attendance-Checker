## Project Name
Student Attendance Checker

## Description
This project is a simple Python program that helps students calculate their attendance percentage and determine whether they are eligible for exams based on attendance rules.

The program takes the total number of classes held and the number of classes attended as input, calculates the attendance percentage, and displays a message about exam eligibility.

## Features
- Accepts user input for total number of classes
- Accepts user input for number of classes attended
- Calculates attendance percentage
- Determines exam eligibility using conditional statements
- Displays a message based on attendance percentage

## Attendance Rules
The program checks attendance according to the following rules:

- **90% or above** → Excellent! Fully eligible for the exam  
- **75% – 89%** → Good! Eligible, but try to attend more classes  
- **50% – 74%** → Warning! Attendance is low  
- **Below 50%** → Not eligible for exams  

## Concepts Used
This beginner-friendly project demonstrates the use of:

- Python Functions  
- Conditional Statements (`if`, `elif`, `else`)  
- User Input  
- Basic Calculations  

## Example
For example, if a student attended **80 classes out of 100 classes**, the program calculates the attendance percentage as **80%** and displays that the student is eligible for the exam.

## How to Run the Program
1. Make sure Python is installed on your computer.
2. Download the file **attendance_checker.py**.
3. Open a terminal or command prompt.
4. Navigate to the folder where the file is saved.
5. Run the program using the command:python attendance_checker.py
6. Enter the total number of classes and the number of classes attended when prompted.

## Sample Output

Enter number of classes attended: 80
